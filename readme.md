# Packwiz for Deno

[![deno module](https://shield.deno.dev/x/packwiz)](https://deno.land/x/packwiz)

An ergonomic Deno wrapper around the CLI tool [packwiz](https://packwiz.infra.link/), used to create and manage Minecraft modpacks.
